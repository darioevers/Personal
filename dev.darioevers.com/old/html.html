    HTML 5 Outlook

    The most interesting "new" HTML5 elements are:

    Semantic elements like:

    • <header>: The header tag in HTML is used to define the header for a document or a section.
    • <footer>: The footer tag in HTML is used to define the footer for a document or a section.
    • <main>: The main tag in HTML is used to define the main content for a document.
    • <nav>: The nav tag in HTML is used to define a set of navigation links.
    • <article>: The article tag in HTML is used to define independent, self-contained content.
    • <section>: The section tag in HTML is used to define a section in a document.
    • <aside>: The aside tag in HTML is used to define content that is aside from the content it is placed in.

    Graphic elements like:

    • <svg>: The svg tag in HTML is used to define graphics for the web.SVG is a vector based graphic.
    • <canvas>: The canvas tag in HTML is used to draw graphics via scripting (usualy JavaScript).

    Multimedia elements like:

    • <audio>: The audio tag in HTML is used to embed sound content in a document.
    • <video>: The video tag in HTML is used to embed video content in a document.

    New types for input elements like number, date, time, week, month, range, color....

    HTML5 API's like Geolocation, Websockets, Offline Storage....

    Learning about HTML elements

    Attributes provide additional information about an element.

    Attributes are always specified in the start tag.

    Attributes usually come in name/value pairs like: name="value".

    Some attributes just have a name with no value.

    Examples:

  • Language: html lang=en-US"
  • Title, Link address, Image attributes: width, heigth, title, alternate text, rel.

    • Hover over me to see the title attribute of this h3 heading.

    (<h3 title="this is the title.">Hover over me to see the title attribute of this h3 heading.</h3>)

    Text formatting elements:

  • <b>= bold text
  • <em>= emphsized text
  • <i>= italic text
  • <u>= underlined text
  • <small>= smaller text
  • <strong>= important text
  • <sub>= subscrupted text
  • <sup>= superscripted text
  • <mark>= marked/highlighted text
  • <del>= deleted text
  • <span>= span/empty element for attributing




    • HTML element as bookmark jump

    Bookmark jumps are useful if your webpage is very long.

    Give the element to jump to an id attribute and refer to that id in your link.

    Example:

    Header Tag with id attribute:

    <h1 id="day1"><u>Day 1</u></h1>

    Text with linked id attribute to jump to webpage part:

    <a href="#day1">Jump to Day 1</a>

    Learning about HTML Form Elements

    The HTML form element defines a form used to collect user input.


    HTML Form Elements elements:

  • <form> Defines a form element
  • <input> Input element, depends on the input type
  • <textarea> Defines a multiline field
  • <label> Defines a label for an <input> field
  • <select> Defines a drop-down list
  • <option> Defines an option in a drop-down list
  • <datalist> List of pre-defined options for an input element

    The HTML Head Element

    Inside the head element, you can for example place the following elements:

  • Title: Defines the page title.
  • Style: Style information for your webpage.
  • Link: Used to link external stylesheets.
  • Meta: Defines meta data like character set, description, keywords, author and more.
  • Script: Defines client side JavaScript for your page.





    • A nested element

    Elements can be nested within other elements. The title for example, is a child element of the head element.
